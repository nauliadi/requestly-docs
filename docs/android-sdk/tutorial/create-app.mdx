Before Integrating the Mobile SDK, you first need to create an app. Follow these steps to create an app.

### 1. Create New App

Go to ['Mobile Debugger'](https://app.requestly.io/mobile-debugger) and then click on the app selector dropdown and then click on Create New App. 

<img alt="Create New App" src="/img/docs/android-sdk/create-app.png" width="60%" />

### 2. Add App Details

Enter `App Name` and `Platform` for the SDK

<img alt="Add Details" src="/img/docs/android-sdk/add-app-details.png" width="60%" />

Your app is now created. You can now follow the rest of the steps for [integration](./integrating-requestly-sdk).
