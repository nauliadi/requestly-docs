### 1. Create Rules

Create rules by navigating to ['HTTP Rules'](https://app.requestly.io/rules/my-rules) in Requestly.

There are many different types of rules available. You can learn about them from here

- [Redirect Requests](https://requestly.io/feature/redirect-url/)
- [Block Requests](https://requestly.io/feature/block-network-requests/)
- [Modify Headers](https://requestly.io/feature/modify-request-response-headers/)
- [Block Request](https://requestly.io/feature/block-network-requests/)
- [Modify Response](https://requestly.io/feature/modify-response/)
- [Modify Query Param](https://requestly.io/feature/modify-query-params/) and many more..


### 2. See if rules are applied or not

Navigate back to[ mobile-interceptor](https://app.requestly.io/mobile-debugger). All the requests that are affected/modified by any rules will get highlighted.
