## Creating Workspace 

In your Requestly app, you can switch your workspace from the side bar. By default, you are in your private workspace. 

<img src="/img/docs/default-ws.png" width="500" align="centre" alt="Workspace Creation" /> <br/> <br/> 


1. Click on the drop down next to <em> Private </em>
2. Go to <em> Create New Workspace </em>
3. Give a name to your Workspace
4. Hit <em> Create New Workspace </em>

<video width="768" height="480" controls>
    <source src="/img/docs/create-ws.mp4" type="video/mp4" />
</video> <br/> <br/> 

Your new Workspace is all set up. <br/> <br/>
