Requestly browser extension helps you **Intercept, Record & Modify HTTP(s) requests directly in the browser**. Available on all major browsers, You can install the browser extension from [the downloads page](https://requestly.io/downloads).
:::tip
Browser extension doesn't require configuring proxy settings & you don't face VPN or SSL issues.
:::
