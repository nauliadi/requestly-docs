1. To Create your first rule, go to [Requestly Dashboard](https://app.requestly.io/rules/create).
2. Select which type of rule you want to create. Learn about different types of rules from [here](../http-modifications/overview).

<figure style={{ width: "75%"}}>
    <img src="/img/docs/rules-list.png" alt="Rules list"/>
    <center><figcaption>Different types of rules</figcaption></center>
</figure>

3. Click on **Create Rule**.
4. Fill in the details for the rule and click on **Create Rule**. Now you are good to go.
