### Why is there a limit of 500 characters for code in the Script Rule?
We use Chrome storage for storing your rules. While everyone loves this choice from Privacy point of 
view, this comes with certain limitations on the size of the object we can store in Chrome Storage. 
You can use Requestly File server to host your JS/CSS/JSON file and pick the file from the Library in your rule editor.
