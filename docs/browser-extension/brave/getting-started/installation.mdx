Since Brave is a chromium-based browser, any chrome extension can be installed on Brave.

To install the Requestly brave extension:
<ol>
    <li>Go to the <a href="https://chrome.google.com/webstore/detail/redirect-url-modify-heade/mdnleldcmiljblolnjhpnblkcekpdkpa">Chrome Web Store</a>.</li>
    <li>Click <b>Add to Brave</b>, then click <b>Add Extension</b>.</li>
    <li>Your extension will be installed.</li>
</ol>
