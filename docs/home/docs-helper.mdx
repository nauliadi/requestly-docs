## Frontmatter
Frontmatter contains all the metadata about the doc
https://docusaurus.io/docs/markdown-features#front-matter

Here are the main ones
```
title: Title of the Page
sidebar_label: Label as seen in the sidebar
description: The description of your document, which will become the <meta name="description" content="..."/> and <meta property="og:description" content="..."/> in <head>, used by search engines.
keywords:
    - Keywords meta tag for the document page, for search engines.
    - Keyword 1
    - Keyword 2
image: Cover or thumbnail image that will be used when displaying the link to your post.
```


[All possible params](https://docusaurus.io/docs/api/plugins/@docusaurus/plugin-content-docs#markdown-front-matter)
