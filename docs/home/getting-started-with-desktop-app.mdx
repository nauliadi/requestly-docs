# Getting Started With Requestly Desktop App

Requestly desktop app helps you **Intercept, Record & Modify HTTP(s) requests for all browsers, Android and iOS apps**. Available on all platforms, You can install the desktop app from [the downloads page](https://requestly.io/downloads).
:::tip
Go to Connected Apps section to launch a browser or Connect to external devices like Android & iOS apps.
:::
