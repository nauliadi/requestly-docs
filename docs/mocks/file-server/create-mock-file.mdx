title: Create New Mock File
---

1. Go to [File Server](https://app.requestly.io/filesv2)
2. Click on **Create New** OR you can **Upload** a JS/CSS/HTML file too.
![Create New Mock API](/img/docs/create-new-file.png)
3. Select the type of file you want to host.
4. Enter the details of the File and then click on **Create**.
    - Name
    - Data
    - Endpoint

![Mock API Editor](/img/docs/file-server-editor.png)


> Your Hosted File is ready now. Copy the URL to see it in action.
