title: Create New Mock API
---

1. Go to [Mocks Dashboard](https://app.requestly.io/mocks)
2. Click on **Create New** OR you can **Upload** a JSON file too.
![Create New Mock API](/img/docs/create-new-mock.png)
3. Enter the details of the Mock API and then click on **Create**.
    - Name
    - Headers
    - Response JSON
    - Endpoint
    - Status Code

![Mock API Editor](/img/docs/mock-api-editor.png)


> Your Mock API is ready now. Copy the URL to see it in action
